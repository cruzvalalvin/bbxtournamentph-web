# Styles

Global styles and CSS utilities.

## Purpose

- Global CSS variables
- Shared style utilities
- Theme configuration
- Animation definitions

## Current Files

- `globals.css` (in app/ directory) - Main global styles

## Guidelines

- Use Tailwind CSS for component styling
- Define CSS variables for theme colors
- Keep global styles minimal
- Use CSS modules for component-specific styles when needed

## Theme Colors

The platform uses a dark metallic theme with amber/gold championship accents:

- Background: `#0a0a0a`
- Card Background: `#111111`
- Championship Gold: `#f59e0b`
- Text: `#ededed`
